# HTTP Basics

## What is HTTP Request?
HTTP request is sent by client to server to ask for data.

Example:
GET /data HTTP/1.1

## What is HTTP Response?
Response is sent by server back to client.

## HTTP Methods
- GET → get data
- POST → send data
- PUT → update data
- DELETE → delete data

## Status Codes
- 200 → OK
- 404 → Not Found
- 500 → Server Error

## Commands used:
`python3 -m http.server 8000`<br>
`curl http://localhost:8000`<br>
