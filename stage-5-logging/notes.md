# Stage 5: Logging System

## What I did

In this stage, I added a logging mechanism to my WAF.

## Why logging is important

Logs help administrators understand what happened in the system and investigate security incidents.

## Implementation

* Processed requests from a file
* Stored request results in a log file
* Logged whether each request was SAFE or BLOCKED

## Outcome

The WAF now maintains a history of processed requests.

## DevOps Concept Learned

Observability and logging are essential parts of production systems.
